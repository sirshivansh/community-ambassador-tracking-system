# Community Ambassador Tracking System

## 📌 Overview
This is a Java Swing-based desktop application designed to manage and track Community Ambassador activities. It allows tracking of student referrals, ambassador performance, and college-wise analytics.

## 🚀 Features
- Add and manage ambassadors
- Prevent duplicate ambassador entries
- Add students with assigned ambassadors
- Generate referral codes (UTM simulation)
- View data in tabular format
- Delete student records
- Search student functionality
- Analytics dashboard:
  - Students per ambassador
  - Top ambassador (with tie handling)
  - Students per college
  - Total summary

## 🛠️ Tech Stack
- Java
- Java Swing (GUI)
- Object-Oriented Programming
- ArrayList (in-memory storage)

## 📊 Key Highlights
- Implemented referral tracking similar to UTM links
- Applied data normalization for consistent analytics
- Built multi-level analytics dashboard
- Designed user-friendly GUI using Swing
